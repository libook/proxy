Features

Raspi gives you a comprehensive collection of HTML, CSS, and JS components
which is simple to use, easy to customize and extendable.

LESS

Raspi is developed in LESS to write well-structured, extendable code which is easy to maintain.

Components

A collection of small, responsive components using consistent and conflict-free naming conventions.

Customizer

Raspi's very basic style can be extended with themes and is easy to customize to create your own look.

Responsive

With the mobile-first approach Raspi provides a consistent experience from phones and tablets to desktops.

Open Source

Raspi is an open source framework to make the web look good.
Create your own styles and discover the unlimited use of Raspi.

MIT License

Raspi is open source and MIT licensed. It is absolutely free of charge and you can use, copy, merge, publish and distribute the framework without any limitations.

Our Heroes

A big thanks to LESS, jQuery, normalize.css and FontAwesome. Raspi wouldn't be possible without these awesome open source projects.

Active Development

Raspi is built by YOOtheme, using our extensive experience in creating Joomla and WordPress themes, to power the Warp theme framework.

Get involved

Raspi is hosted on GitHub and open for everyone to contribute.
Please give us some feedback and join the development!

Contribute

You want to help us and participate in the development or the documentation? Just fork Raspi on GitHub and send us a pull request.

Found a bug?

Open a new issue on GitHub. Please search for existing issues first and make sure to include all relevant information.

Share themes

You created a beautiful Raspi theme? We would love to share it with the community and add it to the Raspi customizer. Send us a pull request.
